# 뽀모도로 타이머 - 요구사항 정의서

> **작성일**: 2026-02-15  
> **작성자**: 소피아 (기획팀)  
> **버전**: v1.0

---

## 1. 프로젝트 개요

### 1.1 배경
- 집중력 관리 도구에 대한 수요 증가
- 기존 뽀모도로 앱들의 복잡함 (불필요한 기능 과다)
- 맥 네이티브 + 플로팅 윈도우 앱 부재

### 1.2 목표
- 심플하고 아름다운 뽀모도로 타이머 제작
- 1인 창업자, 프리랜서 타겟
- MVP 2주 내 완성

---

## 2. 타겟 사용자

### 2.1 주 타겟
| 구분 | 내용 |
|------|------|
| 직업 | 1인 창업자, 프리랜서, 재택근무자 |
| 연령 | 25-45세 |
| 환경 | macOS 사용자 |
| 니즈 | 집중력 관리, 시간 추적 |

### 2.2 페르소나
**"민수" (32세, 1인 개발자)**
- 재택근무, 맥북 프로 사용
- 집중력 유지 어려움, SNS에 쉽게 빠짐
- 복잡한 앱은 싫고, 한눈에 보이는 타이머 원함

## 3. 기능 요구사항

### FR-001: 플로팅 윈도우
| 항목 | 내용 |
|------|------|
| **설명** | 앱 윈도우가 항상 다른 창 위에 표시 |
| **우선순위** | P0 (필수) |
| **수락 기준** | - 다른 앱 위에 항상 표시<br>- 드래그로 위치 이동 가능<br>- 반투명 배경 (블러 효과) |

### FR-002: 타이머 기능
| 항목 | 내용 |
|------|------|
| **설명** | 25분 집중 / 5분 휴식 카운트다운 |
| **우선순위** | P0 (필수) |
| **수락 기준** | - MM:SS 형식 표시<br>- 자동 모드 전환 (집중↔휴식)<br>- 정확한 1초 간격 |

### FR-003: 컨트롤 버튼
| 항목 | 내용 |
|------|------|
| **설명** | 타이머 제어 버튼 |
| **우선순위** | P0 (필수) |
| **수락 기준** | - 시작 버튼 (▶)<br>- 일시정지 버튼 (⏸)<br>- 리셋 버튼 (↺) |

### FR-004: 완료 알림
| 항목 | 내용 |
|------|------|
| **설명** | 세션 완료 시 사용자 알림 |
| **우선순위** | P1 (중요) |
| **수락 기준** | - macOS 시스템 알림<br>- 집중/휴식별 다른 메시지 |

### FR-005: 세션 카운트
| 항목 | 내용 |
|------|------|
| **설명** | 완료한 집중 세션 수 표시 |
| **우선순위** | P1 (중요) |
| **수락 기준** | - 🍅 이모지로 카운트<br>- 앱 재시작 시 리셋 |

## 4. 비기능 요구사항

### NFR-001: 성능
- CPU 사용률 < 1% (백그라운드)
- 메모리 < 50MB
- 배터리 영향 최소화

### NFR-002: 사용성
- 첫 실행 시 바로 사용 가능 (온보딩 없음)
- 드래그로 직관적 위치 조정
- 3초 내 앱 실행

### NFR-003: 디자인
- macOS 네이티브 느낌 (블러, 그림자)
- 미니멀 UI (버튼 3개 + 타이머)
- 시각적 모드 구분 (색상)
