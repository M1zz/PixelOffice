## 1. Executive Summary

### 1.1 서비스 개요
**클립 키보드(ClipKeyboard)**는 사용자가 자주 사용하는 텍스트 정보를 스마트하게 관리하고, iOS 키보드 익스텐션을 통해 어떤 앱에서든 빠르게 입력할 수 있는 **스마트 클립보드 확장 메모 앱**입니다.

### 1.2 핵심 가치 제안 (Value Proposition)
| 가치 | 설명 |
|------|------|
| **시간 절약** | 반복 입력 제거, Combo로 다중 필드 자동 입력 |
| **스마트 정리** | 복사한 내용을 15가지 유형으로 자동 분류 |
| **어디서든 접근** | iOS 키보드 + macOS 메뉴바로 어떤 앱에서든 사용 |
| **안전한 보관** | 생체인증 보호 + iCloud 백업 |

### 1.3 타겟 사용자
- **주 타겟**: 회원가입, 폼 입력이 잦은 20-40대 스마트폰 사용자
- **부 타겟**: 업무상 반복 텍스트 입력이 많은 직장인, 프리랜서

## 2. 시장 분석

### 2.1 문제 정의 (Pain Points)
1. **반복 입력의 번거로움**: 이메일, 전화번호, 주소 등을 매번 타이핑
2. **클립보드 휘발성**: 복사한 내용이 다음 복사로 사라짐
3. **정보 분산**: 중요 정보가 메모, 연락처, 메시지 등에 흩어져 있음
4. **보안 우려**: 민감한 정보(계좌번호 등)를 안전하게 보관할 곳 부재

### 2.2 기존 솔루션의 한계
| 기존 솔루션 | 한계점 |
|-------------|--------|
| 기본 메모 앱 | 키보드에서 바로 접근 불가, 복사-붙여넣기 필요 |
| 클립보드 관리자 | 단순 히스토리, 자동 분류 없음 |
| 자동완성 | 제한된 필드에서만 작동, 커스터마이징 어려움 |

### 2.3 경쟁 우위
- **국내 유일** 스마트 클립보드 자동 분류 (15가지 유형)
- **Combo 시스템**: 여러 정보를 시간 간격으로 순차 입력
- **크로스 플랫폼**: iOS + macOS 완벽 연동
- **네이티브 품질**: SwiftUI 기반 시스템 통합 UX

## 3. 제품 기능 명세

### 3.1 핵심 기능 (Core Features)

#### 3.1.1 스마트 클립보드 (Smart Clipboard)
| 항목 | 상세 |
|------|------|
| **자동 분류** | 복사한 텍스트를 15가지 유형으로 자동 감지 |
| **분류 유형** | 이메일, 전화번호, 주소, URL, 신용카드, 계좌번호, 여권번호, 통관번호, 우편번호, 이름, 생년월일, 세금번호, 보험번호, 차량번호, IP주소 |
| **신뢰도 표시** | 분류 정확도를 0~100%로 표시 |
| **히스토리** | 최대 100개 항목, 임시 항목 7일 자동 삭제 |
| **사용자 수정** | 잘못된 분류 직접 수정 가능 |

#### 3.1.2 메모 관리
| 항목 | 상세 |
|------|------|
| **기본 기능** | 추가, 편집, 삭제, 검색 |
| **즐겨찾기** | 자주 쓰는 메모 상단 고정 |
| **카테고리** | 주제별 분류 (기본, 텍스트, 개인정보 등) |
| **이미지 첨부** | 사진 추가, OCR 텍스트 추출 |
| **사용 통계** | 각 메모별 사용 횟수 기록 |

#### 3.1.3 템플릿 시스템
| 항목 | 상세 |
|------|------|
| **플레이스홀더** | `{변수명}` 형식으로 동적 값 삽입 |
| **자동 제안** | 이전에 입력한 값 히스토리 제공 |
| **사용 예시** | "안녕하세요. {회사명}의 {담당자명}입니다." |

#### 3.1.4 Combo (자동 입력 조합)
| 항목 | 상세 |
|------|------|
| **개념** | 여러 메모/클립을 순차적으로 자동 입력 |
| **시간 간격** | 항목 간 입력 간격 설정 (기본 2초) |
| **상태 표시** | [1/4], [2/4] 형식으로 진행 상황 표시 |
| **사용 예시** | 회원가입 시 이름→이메일→전화번호 순차 입력 |

#### 3.1.5 iOS 커스텀 키보드
| 항목 | 상세 |
|------|------|
| **접근성** | 모든 앱에서 키보드 전환으로 사용 |
| **UI** | 저장된 메모를 버튼 형태로 표시 |
| **필터** | 즐겨찾기, 전체, 카테고리별 필터링 |
| **원탭 입력** | 터치 한 번으로 텍스트 입력 |

#### 3.1.6 macOS 메뉴바 앱
| 항목 | 상세 |
|------|------|
| **메뉴바 아이콘** | 상시 접근 가능한 메뉴바 위치 |
| **전역 단축키** | Control+Option+K (목록), Control+Option+N (새 메모) |
| **클립보드 모니터링** | 자동으로 복사 내용 감지 및 저장 |

#### 3.1.7 보안 기능
| 항목 | 상세 |
|------|------|

| **로컬 저장** | 모든 데이터 기기 내 저장 |

## 4. 사용자 여정 (User Journey)

### 4.1 신규 사용자 온보딩
```
앱 설치 → 온보딩 화면 → 키보드 설정 안내 → 설정 > 키보드에서 활성화 → 완료
```

### 4.2 주요 사용 시나리오

#### 시나리오 1: 회원가입 자동 입력
```
1. 사용자가 회원가입 페이지 접속
2. 키보드를 ClipKeyboard로 전환
3. 저장된 "이메일" 메모 탭 → 자동 입력
4. 다음 필드로 이동, "전화번호" 탭 → 자동 입력
5. (또는 Combo로 한 번에 순차 입력)
```

#### 시나리오 2: 민감 정보 안전 관리
```
1. 계좌번호를 복사 → 앱에서 자동으로 "계좌번호"로 분류
2. "메모로 저장" 선택 → 생체인증 보호 활성화
3. 이후 사용 시 FaceID 인증 후 접근
```

#### 시나리오 3: 업무 템플릿 활용
```
1. "안녕하세요. {회사명}의 {이름}입니다." 템플릿 저장
2. 키보드에서 템플릿 선택
3. 회사명, 이름 입력 → 완성된 문장 자동 입력
```

## 12. 결론

**클립 키보드(ClipKeyboard)**는 단순한 메모 앱을 넘어, 사용자의 반복 입력 작업을 획기적으로 줄여주는 **생산성 도구**입니다.

**차별화 포인트**:
1. **스마트 자동 분류**: 복사만 하면 자동으로 정리
2. **키보드 통합**: 앱 전환 없이 바로 입력
3. **Combo 자동화**: 여러 정보를 한 번에 순차 입력
4. **크로스 플랫폼**: iPhone + Mac 완벽 연동

현재 v3.0.4로 핵심 기능이 완성되어 있으며, 사용자 피드백을 반영한 지속적인 개선이 진행 중입니다.
