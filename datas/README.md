# PixelOffice 데이터 디렉토리

이 폴더는 PixelOffice 앱의 모든 데이터를 저장합니다.

## 📁 구조

```
datas/
├── company.json          # 회사 전체 데이터 (직원, 프로젝트 등)
├── Backups/              # 백업 파일들
├── _shared/              # 전사 공용 데이터
│   ├── documents/        # 전사 공용 문서
│   ├── wiki/             # 회사 위키
│   ├── collaboration/    # 부서 간 협업 기록
│   └── people/           # 전사 직원 프로필
│
└── [프로젝트명]/          # 프로젝트별 데이터
    ├── _shared/          # 프로젝트 공용
    └── [부서명]/          # 부서별 데이터
```

## 🔄 동기화

- **프로젝트 내 저장**: 이 폴더의 모든 데이터는 프로젝트 디렉토리에 저장됩니다
- **Git 동기화**: Git으로 버전 관리되어 다른 컴퓨터에서도 접근 가능합니다
- **iCloud 백업**: company.json은 자동으로 iCloud에도 백업됩니다

## ⚠️ 주의사항

- 민감한 정보(API 키 등)는 이 폴더에 저장하지 마세요
- company.json은 자동 저장되므로 직접 수정하지 마세요
