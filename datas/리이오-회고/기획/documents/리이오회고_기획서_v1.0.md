## 1. 제품 개요

### 1.1 배경

현대인들은 바쁜 일상 속에서 자신을 돌아볼 시간이 부족합니다.
하루가 어떻게 지나갔는지, 무엇을 느꼈는지 기록하지 않으면 금세 잊혀집니다.

**리이오 회고**는 이런 문제를 해결합니다:
- 간편하게 하루를 기록
- 패턴을 발견하고 인사이트 도출
- 개인의 성장을 추적

### 1.2 제품 정의

| 항목 | 내용 |
|------|------|
| **제품명** | 리이오 회고 (Leeo Retrospect) |
| **슬로건** | "하루를 돌아보고, 내일을 준비하다" |
| **제품 유형** | macOS 네이티브 앱 |
| **가격 모델** | 무료 (추후 프리미엄 검토) |

### 1.3 타겟 사용자

**주요 타겟: 자기 성장에 관심 있는 2030 직장인**

| 페르소나 | 특징 | 니즈 |
|----------|------|------|
| **열정 직장인** | 25-35세, 자기계발 관심 | 업무 회고, 성장 기록 |
| **창작자** | 프리랜서, 작가, 개발자 | 아이디어 기록, 영감 정리 |
| **학생** | 대학원생, 취준생 | 학습 기록, 목표 관리 |

## 2. 핵심 기능

### 2.1 일일 회고 기록

**기능 설명**: 매일의 생각과 경험을 기록

**세부 기능**:
- 날짜 선택 (기본: 오늘)
- 자유 텍스트 입력 (마크다운 지원)
- 구조화된 입력 옵션:
  - 🌟 오늘의 하이라이트
  - 🙏 감사한 일
  - 📚 배운 점
  - 💪 개선할 점
- 기분/에너지 레벨 선택 (1-5점)
- 자동 저장

**사용자 스토리**:
> "사용자로서, 퇴근 후 오늘 하루를 5분 안에 기록하고 싶다"

### 2.2 태그 시스템

**기능 설명**: 회고를 주제별로 분류

**세부 기능**:
- # 키워드로 태그 추가
- 기존 태그 자동완성
- 태그별 색상 지정
- 태그 관리 (생성/수정/삭제/병합)

**예시 태그**:
- `#업무` `#개인` `#건강` `#관계` `#학습`

### 2.3 검색 & 탐색

**기능 설명**: 과거 기록을 쉽게 찾기

**세부 기능**:
- 키워드 전문 검색
- 필터 조합:
  - 날짜 범위
  - 태그 선택
  - 기분 점수
- 검색 결과 하이라이트
- 최근 검색어 저장

### 2.4 통계 & 인사이트

**기능 설명**: 패턴 발견과 자기 이해

**세부 기능**:
- 기록 빈도 히트맵 (GitHub 스타일)
- 감정 트렌드 차트
- 자주 사용한 태그 순위
- 워드 클라우드 (자주 쓴 단어)
- 연속 기록일 (스트릭)

### 2.5 데이터 관리

**기능 설명**: 안전한 데이터 보관

**세부 기능**:
- 로컬 저장 (기기 내)
- JSON/마크다운 내보내기
- 백업 파일 생성
- 백업에서 복원

## 3. 사용자 경험

### 3.1 핵심 플로우

```
앱 실행 → 오늘 회고 작성 → 저장 → 완료
         ↓
      (선택) 과거 기록 탐색
         ↓
      (선택) 통계 확인
```

### 3.2 단축키

| 단축키 | 기능 |
|--------|------|
| `⌘ + N` | 새 회고 |
| `⌘ + F` | 검색 |
| `⌘ + S` | 저장 |
| `⌘ + ,` | 설정 |
| `⌘ + 1/2/3` | 탭 전환 |

### 3.3 온보딩

1. 환영 화면 + 앱 소개
2. 첫 회고 작성 유도
3. 태그 추천 안내
4. 완료 및 메인 화면

## 4. 비기능 요구사항

### 4.1 성능

- 앱 실행: 2초 이내
- 검색 응답: 500ms 이내
- 1,000개 회고에서도 버벅임 없음

### 4.2 보안

- 모든 데이터 로컬 저장
- iCloud 동기화 없음 (1차 버전)
- 앱 잠금 기능 (Touch ID / 비밀번호)

### 4.3 접근성

- VoiceOver 지원
- 다이나믹 타입 지원
- 고대비 모드 지원
