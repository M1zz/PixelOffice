## 🛠 구현 내용

### 1. 프로젝트 구조

```
LeeeoRetrospect/
├── LeeeoRetrospect/
│   ├── LeeeoRetrospectApp.swift    # 앱 엔트리포인트
│   ├── Models/
│   │   ├── Retrospect.swift         # 회고 데이터 모델
│   │   └── Tag.swift                # 태그 데이터 모델
│   └── Views/
│       ├── ContentView.swift        # 메인 뷰 (3-column 레이아웃)
│       ├── RetrospectListView.swift # 회고 목록
│       ├── RetrospectDetailView.swift # 회고 상세
│       ├── RetrospectEditorView.swift # 회고 편집기
│       ├── TagListView.swift        # 태그 목록
│       ├── StatsView.swift          # 통계 대시보드
│       └── SettingsView.swift       # 설정
```

### 2. 데이터 모델

#### Retrospect (회고)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | 고유 ID |
| date | Date | 날짜 |
| content | String | 자유 기록 |
| highlight | String? | 오늘의 하이라이트 |
| gratitude | String? | 감사한 일 |
| learned | String? | 배운 점 |
| improvement | String? | 개선할 점 |
| mood | Int | 기분 (1-5) |
| energy | Int | 에너지 (1-5) |
| tags | [Tag] | 태그들 |

#### Tag (태그)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | 고유 ID |
| name | String | 태그 이름 |
| colorHex | String | 색상 (hex) |
| usageCount | Int | 사용 횟수 |

### 3. 주요 기능

- ✅ 회고 CRUD (생성/조회/수정/삭제)
- ✅ 3-column NavigationSplitView 레이아웃
- ✅ 기분/에너지 이모지 선택
- ✅ 구조화된 입력 (하이라이트/감사/배움/개선)
- ✅ 태그 시스템 기본 구현
- ✅ 검색 기능
- ✅ 통계 대시보드 (기본)
- ✅ 설정 화면

### 4. 기술 스택

- **SwiftUI** - UI 프레임워크
- **SwiftData** - 데이터 영속화
- **NavigationSplitView** - macOS 네이티브 레이아웃
