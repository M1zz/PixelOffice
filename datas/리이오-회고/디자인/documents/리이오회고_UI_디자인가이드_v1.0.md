## 4. 레이아웃

### 4.1 앱 구조

```
┌─────────────────────────────────────────────────────┐
│  [사이드바]  │           [메인 콘텐츠]              │
│             │                                       │
│  📅 타임라인 │   ┌───────────────────────────────┐  │
│  🏷 태그     │   │        회고 에디터/뷰어        │  │
│  📊 통계    │   │                               │  │
│  ⚙️ 설정    │   │                               │  │
│             │   └───────────────────────────────┘  │
│             │                                       │
└─────────────────────────────────────────────────────┘
```

### 4.2 사이드바

- 너비: 220pt (고정)
- 배경: Secondary Background
- 아이템 높이: 32pt
- 아이콘 크기: 16pt
- 선택 상태: Accent 색상 배경

### 4.3 메인 에디터

- 최대 너비: 720pt (읽기 편안함)
- 좌우 패딩: 40pt
- 상하 패딩: 32pt

## 5. 컴포넌트

### 5.1 회고 카드

```
┌──────────────────────────────────────┐
│ 2월 15일 토요일                      │
│ 😊 좋음                              │
├──────────────────────────────────────┤
│ 오늘 PixelOffice 기능을 많이 추가    │
│ 했다. 뿌듯하다...                    │
├──────────────────────────────────────┤
│ #개발 #PixelOffice #성취             │
└──────────────────────────────────────┘
```

- 모서리: 12pt radius
- 패딩: 16pt
- 그림자: 미세한 drop shadow (다크모드에서는 없음)

### 5.2 태그 칩

```
[ #업무 ] [ #개인 ] [ + ]
```

- 높이: 24pt
- 패딩: 8pt 좌우
- 모서리: 12pt radius (pill shape)
- 색상: 태그별 지정 or 기본 Accent

### 5.3 감정 선택기

```
😢  😔  😐  🙂  😊
 1   2   3   4   5
```

- 이모지 크기: 24pt
- 선택 시: 배경 원형 + 스케일 업

### 5.4 통계 히트맵

- GitHub 스타일 기여 그래프
- 셀 크기: 12x12pt
- 간격: 2pt
- 색상 강도: 기록 횟수에 따라 4단계
