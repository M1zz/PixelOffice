## 🎯 프로젝트 비전

**"하루를 돌아보고, 내일을 준비하는 나만의 공간"**

바쁜 일상 속에서 잠시 멈추고 오늘을 돌아볼 수 있는 회고 앱.
간단한 기록부터 깊은 성찰까지, 나의 성장을 기록하고 추적합니다.

## ✨ 핵심 기능

### 1. 일일 회고 기록
- 날짜별 회고 작성
- 오늘의 기분/에너지 레벨 기록
- 감사한 일, 배운 점, 개선할 점 섹션

### 2. 태그 시스템
- 자유로운 태그 추가
- 태그별 필터링/검색
- 자주 쓰는 태그 추천

### 3. 검색 & 탐색
- 키워드 전문 검색
- 날짜 범위 필터
- 태그 조합 검색

### 4. 통계 & 인사이트
- 주간/월간 회고 빈도
- 감정 트렌드 그래프
- 자주 등장하는 키워드

### 5. 데이터 관리
- 로컬 저장 (CoreData/SwiftData)
- 백업/복원 기능
- 마크다운 내보내기
