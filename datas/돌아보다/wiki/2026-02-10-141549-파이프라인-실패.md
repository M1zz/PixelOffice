## 📝 요구사항

> ### 🎯 제품 비전                                                                                              
                                                                                                               
 "하루 5분, 성장을 기록하는 습관"                                                                              
                                                                                                               
 매일/매주 간단하게 회고를 작성하고, 지난 기록을 돌아보며 성장 패턴을 파악하는 개인 회고 앱                    
                                                                                                               
 ────────────────────────────────────────────────────────────────────────────────                              
                                                                                                               
 ### 👤 타겟 사용자                                                                                            
                                                                                                               
 - 자기 성장에 관심 있는 직장인                                                                                
 - 매일 업무를 돌아보고 싶은 사람                                                                              
 - 간단한 회고 습관을 만들고 싶은 사람                                                                         
                                                                                                               
 ────────────────────────────────────────────────────────────────────────────────                              
                                                                                                               
 ### 🔧 핵심 기능                                                                                              
                                                                                                               
 #### 1. 회고 작성                                                                                             
                                                                                                               
 ┌─────────────┬─────────────────────────────────────┐                                                         
 │ 항목        │ 설명                                │                                                         
 ├─────────────┼─────────────────────────────────────┤                                                         
 │ 기간 선택   │ 일간 / 주간 선택                    │                                                         
 ├─────────────┼─────────────────────────────────────┤                                                         
 │ 완료한 일   │ 오늘/이번 주 완료한 일 3가지 (필수) │                                                         
 ├─────────────┼─────────────────────────────────────┤                                                         
 │ 잘한 점     │ Keep - 계속할 것                    │                                                         
 ├─────────────┼─────────────────────────────────────┤                                                         
 │ 아쉬운 점   │ Problem - 개선할 것                 │                                                         
 ├─────────────┼─────────────────────────────────────┤                                                         
 │ 다음 목표   │ Try - 다음에 시도할 것              │                                                         
 ├─────────────┼─────────────────────────────────────┤                                                         
 │ 한 줄 소감  │ 오늘/이번 주를 한마디로             │                                                         
 ├─────────────┼─────────────────────────────────────┤                                                         
 │ 기분 이모지 │ 😊😐😢😤🔥 중 선택                  │                                                         
 └─────────────┴─────────────────────────────────────┘                                                         
                                                                                                               
 #### 2. 히스토리 보기                                                                                         
                                                                                                               
 - 날짜별 회고 목록 (캘린더 뷰 또는 리스트 뷰)                                                                 
 - 회고 상세 보기                                                                                              
 - 검색 (키워드로 과거 회고 찾기)                                                                              
                                                                                                               
 #### 3. 통계 & 인사이트                                                                                       
                                                                                                               
 - 연속 회고 일수 (스트릭)                                                                                     
 - 월별 기분 추이 그래프                                                                                       
 - 자주 언급한 키워드 (워드클라우드)                                                                           
                                                                                                               
 #### 4. 데이터 관리                                                                                           
                                                                                                               
 - 로컬 저장 (SwiftData 또는 JSON 파일)                                                                        
 - 백업/복원 (파일 내보내기/가져오기)                                                                          
                                                                                                               
 ────────────────────────────────────────────────────────────────────────────────
